# Password_Strength_Checker
A full-stack Password Strength Analyzer that evaluates password security in real-time using Python and Flask. The application checks password strength based on length, character diversity, and common password patterns, and provides instant feedback, entropy score, crack-time estimation, and improvement suggestions

🚀 Features
Real-time password strength analysis
Score-based strength meter (0–5 scale)
Entropy & crack-time estimation
Live requirement checklist
Password statistics breakdown
Common password detection
Password generator
Show/Hide password toggle
Dark/Light mode support
Fully responsive UI
